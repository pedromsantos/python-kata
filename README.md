## Running Tests

To run the tests for the various katas, you can use the following commands:

- **All Tests**: Run all tests with the command:

  ```bash
  make test
  ```

### Individual Tests

You can also run tests for individual katas using the following commands:

- **Fizz Buzz**:

  ```bash
  make fizz
  ```

- **Fibonacci**:

  ```bash
  make fib
  ```

- **Leap Year**:

  ```bash
  make leap
  ```

- **Prime Factors**:

  ```bash
  make prime
  ```

- **Roman Numerals**:

  ```bash
  make roman
  ```

- **Stack Kata**:

  ```bash
  make stack
  ```

- **Tic Tac Toe**:

  ```bash
  make tic
  ```

- **Yahtzee**:

  ```bash
  make yahtzee
  ```

- **Tennis**:

  ```bash
  make tennis
  ```

- **Gilded Rose**:


- **Raid**:

- **Smelly Tic Tac Toe**:

- **ESA Mars Rover**:

- **Social Network**:

- **London Tic Tac Toe**:

- **Raid**:
